Inside every phase folder, keep the same four: Contracts, RFIs & Submittals, Change Orders, Daily Logs. Same layout in each phase — so you always know where a doc lives.

Why It Works

1Folders = the schedule

Phases match the Gantt, so files land where the work is and the order never shifts.

2Same four in every phase

Contracts · RFIs · Change Orders · Daily Logs — identical inside each phase folder.

3Date files inside-out

Lead with YYYY-MM-DD so files auto-sort by date within a folder.

4Photos stay together

One _Photos folder for all phases — easier to scan progress over time.
